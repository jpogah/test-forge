# Forge Hello World Server

A simple Go HTTP server that responds with "Hello from Forge".

## Prerequisites

- [Go](https://go.dev/) 1.18 or later

## Running the server

```bash
go run main.go
```

The server starts on port 8080. Visit [http://localhost:8080/](http://localhost:8080/) to see the response.

## Testing

```bash
curl http://localhost:8080/
# Output: Hello from Forge
```
